# База данных «Контроль домашних заданий»

## Описание
Проект для курсовой работы. База данных для учёта домашних заданий, сданных работ и оценок.

## Таблицы
- Students (ученики)
- Teachers (учителя)
- Subjects (предметы)
- Homeworks (задания)
- Submissions (сданные работы)
- Grades (оценки)
